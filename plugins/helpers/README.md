# helpers

General improvements to Claude's behavior. This plugin contains foundational skills that make Claude more reliable and accurate.

## Skills

### /helpers:verifying-day-of-week

Instructs Claude to use the system `date` command instead of calculating the day of the week itself. Claude will auto-invoke this skill whenever it needs to know or state a day of the week.

## Installation

```
/plugin install helpers@ruddiments
```
