# Lookup Day of Week Reference

When you need to know or state a day of the week, **do not calculate it yourself**. Instead, run:

```bash
date +"%A, %B %d, %Y"
```

This returns the authoritative day of the week from the system clock (e.g., "Saturday, January 24, 2026").
